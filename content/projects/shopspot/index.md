### Description
ShopSpot App is a lifestyle-oriented platform that integrates the 3C Model – Content, Commerce, and Community to better respond to users’ needs and evolving behavior. The app is designed with a modular architecture combining MVC and MVP patterns, enabling maintainable, scalable, and testable code. It leverages modern Android development practices to deliver a smooth, responsive, and user-centric experience.

### Tech Stack
- **MVP (Model–View–Presenter)** – promotes a reactive and testable presentation layer.
- **Java** – powers the core Android development with compatibility for legacy components.
- **RxJava** – enables reactive programming and asynchronous event handling.
- **REST API** – enables efficient communication with backend services.

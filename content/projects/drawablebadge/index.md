This library is an Android utility written in Kotlin that provides a simple and efficient way to add badges to drawables. It is designed for developers who need to display notification counts, status indicators, or labels directly on icons or images without manually handling canvas drawing or layout overlays. The library supports custom badge positions, colors, and text styles, making it flexible for various UI.

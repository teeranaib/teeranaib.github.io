### Description
WPB Pocket App is a mobile solution for property management company managers, designed to streamline asset tracking, task management, and communication. The app employs an offline-first MVVM architecture, ensuring reliability and seamless functionality even in low-connectivity environments. Built with a modular and maintainable architecture, it leverages modern Android technologies to deliver a robust, responsive, and efficient user experience.

### Tech Stack
- **Kotlin** – provides concise, expressive, and type-safe Android development.
- **Hilt** – simplifies dependency injection for clean, modular code.
- **Room** – enables efficient local database storage for offline functionality.
- **Coroutines** – supports asynchronous programming for smooth user interactions.
- **WebSocket** – allows real-time data synchronization and updates.
- **REST API** – provides reliable backend communication.
- **Jetpack Compose** – delivers modern, declarative UI development.
- **Offline-first MVVM architecture** – ensures offline reliability while maintaining clean separation of concerns.

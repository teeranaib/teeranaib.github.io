### Description
Yay! App is a social networking platform that connects users of the same generation nationwide through chat and voice calls. The app is built with a modular and maintainable MVVM architecture, providing real-time communication, smooth UI interactions, and scalable performance. It leverages modern Android technologies to deliver an intuitive and responsive user experience.

### Tech Stack
- **MVVM (Model–View–ViewModel)** – ensures clean separation between UI and business logic.
- **Kotlin** – enables concise, expressive, and type-safe Android development.
- **Jetpack Compose** – allows modern, declarative UI construction with reactive updates.
- **Koin** – simplifies dependency injection and module management.
- **Coroutines & RxJava** – support asynchronous programming and reactive data flows.
- **Realm** – provides an efficient and reliable local database for offline and real-time data management.

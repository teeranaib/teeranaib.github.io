### Description
WealthPark App is a real estate investment management platform that allows users to monitor, analyze, and manage property assets seamlessly across devices. The app was migrated from a legacy Java codebase to Kotlin, enabling more concise, expressive, and maintainable development. It delivers real-time updates and secure communications while leveraging a scalable and modular architecture for long-term maintainability and performance.

### Tech Stack
- **MVVM (Model–View–ViewModel)** – ensures structured presentation logic and maintainable UI interactions.
- **Kotlin & Java** – supports modern development while maintaining compatibility with the existing codebase.
- **Hilt** – handle dependency injection efficiently across modules.
- **RxJava & Coroutines** – enable reactive and asynchronous data handling.
- **WebSocket** – provides real-time data synchronization for investment updates.
- **REST API & GraphQL** – allow flexible and efficient communication with backend services.
- **Jetpack Compose** – enables modern, declarative UI development with enhanced performance.

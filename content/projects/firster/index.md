### Description
Firster is a modern lifestyle shopping application that offers a wide range of unique and innovative products through an intuitive and visually engaging user experience. The app is built with a modular and maintainable architecture, focusing on scalability, clean code organization, and efficient development workflows.

### Tech Stack
- **Clean Architecture** – ensures clear separation of concerns and maintainable code structure.
- **MVVM (Model-View-ViewModel)** – promotes a reactive and testable presentation layer.
- **Kotlin** – provides concise, expressive, and safe language features for Android development.
- **Hilt** – simplifies dependency injection and promotes modularized dependency management.
- **Coroutines** – enables efficient asynchronous and concurrent programming.
- **Android App Modularization** – improves build performance, feature isolation, and reusability.
- **REST API** – enables efficient communication with backend services.

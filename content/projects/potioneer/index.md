### Description
Potioneer App is a mobile platform that enables users to discover and book intimate private dining experiences crafted by professional chefs. Each course menu is uniquely curated to offer personalized and memorable culinary journeys. The app is developed with a scalable, maintainable, and modular architecture, ensuring smooth performance and flexibility for continuous feature evolution.

### Tech Stack
- **MVVM (Model–View–ViewModel)** – provides a clear separation of concerns and reactive data flow.
- **Kotlin** – delivers concise, expressive, and type-safe Android development.
- **Hilt** – simplifies dependency injection for clean and modular code.
- **RxJava** – supports reactive programming and asynchronous event handling.
- **REST API** – enables efficient communication with backend services.
- **Epoxy** – streamlines dynamic UI construction with a declarative RecyclerView framework.
